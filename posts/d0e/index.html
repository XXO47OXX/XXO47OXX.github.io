试题 算法训练 共线

题目链接

资源限制

时间限制：1.0s 内存限制：256.0MB

问题描述

　　给定2维平面上n个整点的坐标，一条直线最多能过几个点？

输入格式

　　第一行一个整数n表示点的个数
　　以下n行，每行2个整数分别表示每个点的x,y坐标。

输出格式

　　输出一个整数表示答案。

样例输入

5
0 0
1 1
2 2
0 3
2 3

样例输出

3

数据规模和约定

　　n<=1500，数据保证不会存在2个相同的点。
　　点坐标在int范围内

问题分析

1、直线的斜率的取值范围是:-∞<k<+∞,斜率k可以是一切实数,k∈R

2、两点确定一条直线

3、题目所求的是“一条直线上最多的点”，那么这条直线上最少会有两个点，反向思考→一个点在拥有最多点的直线上，那么它就会拥有许多和其它点相同的斜率。也就是说，在一个点所拥有的最高频次的斜率即为，题目所求的解。

4、如要存储斜率信息，需要考虑到斜率不存在的情况，即，点落在横轴上

算法设计

1、先用二维数组存储坐标信息，point [i] [0]存储x坐标，point [i] [2]存储y坐标。第一个双重for循环录入坐标信息。

2、第二个双重for循环，计算每个点与其它点所连成的直线的斜率，并计算每个斜率的频次。此处设置一个整型数other来记录斜率不存在的情况。

3、每计算一个point[i]和point[j]的斜率，则将数据存储至point[j] [2]中，并用point[j] [3]记录其数据出现频次，每记录一个频次A，就往前找是否有重复值B出现，并对两者的频次进行相加，即A+B。

4、确定频次后及时同max进行比较记录最大值。

5、确定最大值后，可用if（max>n-i-1）判断是否提前结束计算。因为每计算过一轮i值，i值对应的那个点C与C对应的所以斜率都已无意义，可以进行数据覆盖，且不再参与计算。当max大于下一轮斜率数，答案就是当前的max值。

实现代码

#include <iostream>
using namespace std;
int main()
{
    int n;
    cin>>n;
    int max=1;
    double point[1500][4]= {0};
    for(int i=0; i<n; i++)
    {
        cin>>point[i][0]>>point[i][1];
    }//记录点坐标信息
    for(int i=0; i<n; i++)
    {//计算过点斜率
        int other=0;
        for(int j=i+1; j<n; j++)
        {
            //从i开始遍历点计算直线
            int y=(point[j][0]-point[i][0]);
            if(y!=0)
            {
                point[j][2]=((point[j][1]-point[i][1])/y);
                point[j][3]=1;
                for(int k=j-1; k>i; k--)
                {
                    if(point[k][2]==point[j][2])
                    {
                        point[j][3]+=point[k][3];break;
                    }
                }
            }
            else
            {
                point[j][3]=0;
                other++;//不存在的斜率次数计算
            }//计算过该点的各斜率出现的次数；
            if((point[j][3]+1)>max)
                max=point[j][3]+1;
        }
        if(other>max)
                    max=other;
        if(max>n-i-1)
            break;
    }
    cout<<max<<endl;
    return 0;
}
