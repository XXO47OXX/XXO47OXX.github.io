找出唯一成对的数

1-1000这1000个数放在含有1001个元素的数组中，只有唯一的一个元素值重复，其它均只出现一次。每个数组元素只能访问一次，设计一个算法，将它找出来;不用辅助存储空间，能否设计一个算法实现?

※异或：不进位加法

​ 性质

​ 1、交换律可任意交换运算因子的位置，结果不变

​ 2、结合律(即(a^b)^c == a^(b^c))

​ 3、对于任何数x，都有x^x=0，x^O=x，同自己求异或为0，同0求异或为自己

​ 4、自反性A^B^B=A^0=A，连续和同一个因子做异或运算，最终结果为自己

new Random().nextInt(n)

​ 该方法的作用是生成一个随机的int值，该值介于[0,n)的区间，也就是0到n之间的随机int值，包含0而不包含n。

读题解代码：

import java.util.Random;


public class hell {
    /**
     * 交换数组中的两个数,一个数自己与自己异或结果为0；一个数与0异或，结果还是自己。
     */
    public static void swap(int[] arr, int i, int j) {
        arr[i] = arr[i] ^ arr[j];
        arr[j] = arr[i] ^ arr[j];
        arr[i] = arr[i] ^ arr[j];
    }
    public static void main(String[] orgs) {
        int N=11;//设置数组长度
        int[] arr = new int[N];//定义长度为N+1的数组
        for (int i = 0; i < arr.length-1; i++) {
            arr[i]=i+1;
        }//初始化N-1的数组

//最后一个数，是随机数
        arr[arr.length-1]=new Random().nextInt( N-1)+1;//最后一个数为1~N-1随机数
        int index = new Random().nextInt(N);//随机下标
        swap(arr,index,arr.length-1);//交换位置
        for(int i=0;i<=N-1;i++) {
            System.out.print(" ["+arr[i]+"]");
        }
        System.out.println(" ");
        int x1=0;//任何数和0异或都为其本身
        for(int i=0;i<=N-1;i++) {
            x1 = (x1^(i));
        }//因为已知arr中为[1,N-1]连续整数+一个唯一重复数，arr异或[1,N-1]结果则为那个唯一重复数，该循环的目的是得到[1,N-1]中的各个数异或结果x1
        for(int i=0;i<=N-1;i++) {
            x1 = (x1^(arr[i]));
        }//该循环的目的是将arr数组中的各个数进行连续异或，并将异或结果同x1进行异或
        System.out.println("该数组的唯一重复数为："+x1);
    }
}

解题核心：

​ 1、已知数串结构为 一组已知连续整数+已知连续整数中的一个数

​ 2、利用A^A=0,A^0=A(同自己求异或为0，同0求异或为自己)

​ 3、自反性A^B^B=A^0=A，连续和同一个因子做异或运算，最终结果为自己
