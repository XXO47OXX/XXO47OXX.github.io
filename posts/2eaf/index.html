一、相关基础

  1. 位运算符
  • 在处理整形数值时，可以直接对组成整形数值的各个位进行操作。这意味着可以使用屏蔽技术获得整数中的各个位（? ? )
  • &(与)、|(或)、^(异或)、~(非/取反)
  • >>和<<运算符将二进制位进行右移或者左移操作
  • >>>运算符将用0填充高位;>>运算符用符号位填充高位，没有<<<运算符
  • 对于int型，1<<35与1<<3是相同的，而左边的操作数是long型时需对右侧作数模64(因为int的长度只有32位
  • 与:都为1结果为1，或:有一个为1结果为1，异或:二者不同时结果为1

  1. 位运算符奇巧淫技

    1. 判断奇偶数

      1. x&1=1奇0偶

    3. 获取二进制位是1还是0（两种）

      1. 将1左移后同数值做与运算同（a）

    5. 交换两个整数变量的值

      1. a = a^b;

        b = a^b;

        a = a^b;（a,b不相等）

    6. 不用判断语句，求整数的绝对值

      (num^(num>>31))+(num>>>31)

11111111 11111111 11111111 11110111：num

11111111 11111111 11111111 11111111：num>>31

00000000 00000000 00000000 00001000：num^(num>>31)

00000000 00000000 00000000 00000001：num>>>31

00000000 00000000 00000000 00001001：
 (num^(num>>31))+(num>>>31)
正数显然满足
负数的相反数是原码含符号位取反加一

    7. x&(x-1) ：可以使二进制数x的最低位的1变成0；

    8. 异或，不进位加法

  3. 性质

1、交换律可任意交换运算因子的位置，结果不变

2、结合律(即(a^b)^c == a^(b^c))

3、对于任何数x，都有x^x=0，x^0=x，同自己求异或为0，同0求异或为自己

4、自反性A^B^B=A^0=A，连续和同一个因子做异或运算，最终结果为自己

  1. 找出唯一成对的数

    1. 1-1000这1000个数放在含有1001个元素的数组中，只有唯一的一个元素值重复，其它均只出现一次。每个数组元素只能访问一次，设计一个算法，将它找出来;不用辅助存储空间，能否设计一个算法实现?

      1. 强行配凑消去其他元素：这1001个元素连续异或，再异或上1-1000这1000个数。

    3. 其他元素都出现2次，异或后变成0.而唯一的一个元素值出现3次，连续异或后值不变。

public static void main(String[] args) {

​    int []arr = new int[]{1,2,3,4,4,5};

​      ans^=i;

  4.   public static void  main(String[] args)
   
{    

int a =  0b01000000_00000000_00000000_00000000;

System.out.println(a);

System.out.println(b); //0b10000000_00000000_00000000_00000000

private static int m(int i) {    

int ou = i & 0xaaaaaaaa;//和1010  1010 1010 ……做与运算取出偶数位

int ji = i & 0x55555555;//和0101  0101 0101 ……做与运算取出奇数位

return (ou >>> 1) ^ (ji <<  1); // 连起来   }
